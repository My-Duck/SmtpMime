Скрипт, отправляющий e-mail со всеми изображениями из папки в качестве вложения
Использование: python main.py -s server -t to
Дополнительные флаги:
-f --from: e-mail отправителя (по умолчанию <>)
--subject: тема сообщения (по умолчанию Happy Pictures)
--auth: авторизация на сервере
-v --verbose: показывать запросы и ответы сервера
-d --directory директория с изображениями (по умолчанию $pwd)
-ssl: использовать ssl
-p --pipeline: использовать pipelining (если сервер поддерживает)
